## Recent Updates

Stay informed about new features, bug fixes, improvements, and compliance changes in Tima. You receive notifications in your dashboard when updates deploy. Review the details below to understand impacts on your workflows.

<Update label="2024-10-20" description="v2.3.0" tags={["feature", "improvement", "compliance"]}>

## New Features

- Added real-time portfolio valuation with live market data feeds from major exchanges.
- Introduced compliance dashboard for tracking regulatory filings and audit trails.

## Improvements

- Enhanced investment recommendation engine with AI-driven risk assessments.
- Optimized mobile app performance, reducing load times by `>40%`.

## Bug Fixes

- Fixed intermittent sync issues between dashboard and `https://api.example.com/v2/portfolios`.
- Resolved display errors in multi-currency transaction histories.

## Compliance Updates

- Updated KYC verification to align with latest EU MiFID II requirements.
- Added support for new tax reporting formats in APAC regions.

</Update>

<Update label="2024-09-25" description="v2.2.1" tags={["bugfix", "security"]}>

## Bug Fixes

- Patched vulnerability in OAuth token refresh endpoints (security advisory TIMA-2024-001).
- Corrected calculation errors in compound interest projections for long-term investments.

## Improvements

- Streamlined webhook delivery retries, now supporting up to 10 attempts with exponential backoff.
- Improved search functionality in transaction logs with fuzzy matching.

## Security Updates

- Mandated two-factor authentication for all high-value withdrawals `>10,000 USD`.

</Update>

<Update label="2024-09-10" description="v2.2.0" tags={["feature", "breaking"]}>

## New Features

- Launched beta for automated investment portfolios with customizable strategies.
- Integrated third-party wallet connections for seamless crypto transfers.
