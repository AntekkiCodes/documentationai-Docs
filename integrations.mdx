## Link Bank Accounts

Link external bank accounts to Tima to track balances and transactions automatically.

<Callout kind="info">
  Ensure your bank supports Open Banking APIs like Plaid or TrueLayer.
</Callout>

<Steps>
  <Step title="Get API Credentials" icon="key">
    Sign up for your bank's developer portal and obtain `{CLIENT_ID}` and `{CLIENT_SECRET}`.
  </Step>
  <Step title="Initiate Linking" icon="link">
    Use Tima's endpoint to start the connection process.
  </Step>
  <Step title="Verify Access" icon="check-circle">
    Confirm the link and test data retrieval.
  </Step>
</Steps>

<CodeGroup tabs="JavaScript,cURL">
  ```javascript
  const response = await fetch('https://api.example.com/v1/accounts/link', {
    method: 'POST',
    headers: { 'Authorization': `Bearer ${YOUR_API_KEY}` },
    body: JSON.stringify({
      bankId: 'plaid-us-bank',
      clientId: 'YOUR_CLIENT_ID',
      clientSecret: 'YOUR_CLIENT_SECRET'
    })
  });
  ```
  ```bash
  curl -X POST https://api.example.com/v1/accounts/link \
    -H "Authorization: Bearer YOUR_API_KEY" \
    -H "Content-Type: application/json" \
    -d '{
      "bankId": "plaid-us-bank",
      "clientId": "YOUR_CLIENT_ID",
      "clientSecret": "YOUR_CLIENT_SECRET"
    }'
  ```
</CodeGroup>

## Webhook Setups

Set up webhooks to receive real-time updates on transactions and account changes.

<Request tabs="JavaScript,Python" show-lines="true">
  ```javascript
  // Register webhook endpoint
  await fetch('https://api.example.com/v1/webhooks', {
    method: 'POST',
    headers: { 'Authorization': `Bearer ${YOUR_API_KEY}` },
    body: JSON.stringify({
      url: 'https://your-webhook-url.com/tima',
      events: ['transaction.created', 'account.updated']
    })
  });
  ```
  ```python
  import requests
  response = requests.post(
    'https://api.example.com/v1/webhooks',
    headers={'Authorization': f'Bearer {YOUR_API_KEY}'},
    json={
      'url': 'https://your-webhook-url.com/tima',
      'events': ['transaction.created', 'account.updated']
    }
  )
  ```
</Request>

Validate incoming payloads with Tima's HMAC signature.

```javascript
app.post('/tima', (req, res) => {
  const signature = req.headers['tima-signature'];
  const isValid = verifySignature(req.body, signature, YOUR_WEBHOOK_SECRET);
  if (isValid) {
    // Process event
    res.status(200).send('OK');
  }
});
```
